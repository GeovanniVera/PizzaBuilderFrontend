# La Toscana — Frontend

Aplicación de punto de venta para La Toscana, con 3 secciones: Vender, Catálogo, Corte de caja.

## Cómo correrlo

```bash
npm install
npm run dev
```

Abre http://localhost:5174

## Backend

Espera que el backend esté en http://localhost:8080.
